Accountant Assistant

A private bookkeeping helper, run locally by the business owner.

Effective date: June 17, 2026

Privacy Policy

This policy describes how Accountant Assistant handles data. It is written to be honest and accurate to what this specific tool actually does.

Who operates this tool

Accountant Assistant is operated by the business owner for their own internal use, with their own QuickBooks Online companies. It is a single-operator tool that is not sold to or used by anyone else. Questions can be sent to jmeister53@gmail.com.

What data is accessed

The tool accesses the owner's own QuickBooks Online accounting data — for example the chart of accounts (the list of categories), transactions, and standard reports. This is accessed through Intuit's official connection (its public interface for QuickBooks Online), with the owner's authorization. The tool does not collect data from any other person or any other source.

How data is accessed

The connection uses the standard, secure sign-in method that Intuit provides (OAuth 2.0), where the owner approves access through Intuit's own login screen. The tool's normal mode is read-only — it reads the books to review them. The only changes it ever makes are corrections the owner reviews and approves one at a time. Nothing is changed automatically.

How data is stored

All data stays on the owner's own computer, in local files on that machine. The connection credentials (the keys that let the tool reach QuickBooks Online) are kept in a local environment file on the same computer. This information is not transmitted to, sold to, or shared with any third party. The tool uses no analytics and no tracking of any kind.

Security

Secrets and connection credentials are stored only in a local file that is kept out of any shared or published code (it is git-ignored). Tokens and credentials are never logged and never shared.

Data retention and deletion

Because the data lives only on the owner's own computer, the owner controls it entirely. The owner can delete the data at any time by removing the local files and disconnecting the tool from QuickBooks Online. See How to disconnect for step-by-step instructions.

Not affiliated with Intuit

Accountant Assistant is not affiliated with, sponsored by, or endorsed by Intuit Inc. QuickBooks and QuickBooks Online are trademarks of Intuit Inc.
